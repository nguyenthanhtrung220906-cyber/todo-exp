# ⚡ Todo EXP — Desktop App

A gamified productivity app inspired by Microsoft To Do.
Built with **Electron + Node.js**. Runs completely offline.

---

## 📁 Project Structure

```
todo-exp-desktop/
│
├── main.js              ← Electron main process (window + IPC + storage)
├── preload.js           ← Secure contextBridge API for renderer
├── package.json         ← Dependencies + electron-builder config
│
├── renderer/
│   ├── index.html       ← App shell + title bar + EXP bar
│   ├── app.js           ← All UI logic (tasks, lists, EXP, drag-drop)
│   └── style.css        ← Full polished styles (dark mode + animations)
│
├── assets/
│   ├── icon.ico         ← Windows icon (multi-size: 16,32,48,64,128,256)
│   ├── icon.png         ← 256×256 PNG icon
│   └── license.txt      ← License shown in installer
│
└── dist/                ← Generated by npm run dist (gitignored)
    └── TodoEXP-Setup-1.0.0.exe
```

## 🚀 Quick Start (Development)

### Prerequisites
- **Node.js** 18+ → https://nodejs.org
- **npm** 9+
- **Windows** (for building .exe) or any OS for development

### 1. Install dependencies

```bash
cd todo-exp-desktop
npm install
```

This installs:
- `electron` — Desktop runtime
- `electron-builder` — Installer generator

### 2. Run in development mode

```bash
npm start
```

The app opens immediately. DevTools open automatically in dev mode:

```bash
npm run dev
```

## 📦 Build Windows Installer

### Generate Setup.exe

```bash
npm run dist
```

This runs `electron-builder --win --x64` and produces:

```
dist/
└── TodoEXP-Setup-1.0.0.exe   ← Ready to distribute!
```

### Build app folder only (faster, no installer)

```bash
npm run dist:dir
```

## 💾 Data Storage

Task data is saved as JSON at:

```
Windows: C:\Users\<You>\AppData\Roaming\todo-exp\todoexp-data.json
```

- **Auto-saves** 300ms after every change (debounced)
- **Atomic writes** (write to .tmp → rename) to prevent corruption
- **Persists** across app restarts and system reboots
- You can open the data folder from within the app (📁 button in sidebar)

## 🎮 EXP System

- Each task has an **EXP value** (default: 10)
- Completing a task adds EXP → floating "+10 EXP" animation
- Unchecking removes EXP
- Every **100 EXP** = 1 Level up
- Level shown in bottom EXP bar + header badge

---

## 🔒 Security

- `contextIsolation: true` — renderer isolated from Node.js
- `nodeIntegration: false` — no Node APIs in renderer
- All Node/OS access goes through `preload.js` contextBridge
- CSP headers block remote code execution

## 🛠️ Troubleshooting

**"npm install" fails on Windows?**
→ Run as Administrator, or install `windows-build-tools`:
```bash
npm install --global windows-build-tools
```

**Icon not showing in installer?**
→ Ensure `assets/icon.ico` is a valid multi-size ICO file.

**App doesn't start?**
→ Check `npm start` output. Make sure Electron version matches Node.js version.

**Data not persisting?**
→ Check `%APPDATA%\todo-exp\` exists and is writable.
